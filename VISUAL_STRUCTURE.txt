═══════════════════════════════════════════════════════════════════════════════
                             ANIMATION FLOW
═══════════════════════════════════════════════════════════════════════════════

SCROLL DOWN
    │
    ├─► Header: Transparent → Glass Effect
    ├─► Background: rgba(30,15,17,0) → rgba(30,15,17,0.7)
    ├─► Border: Hidden → Visible
    └─► Shadow: None → Premium Shadow
    
HOVER MENU ITEM
    │
    ├─► Underline: scaleX(0) → scaleX(1)
    ├─► Color: White → Red
    ├─► Chevron: rotate(0deg) → rotate(180deg)
    └─► Glow: opacity(0) → opacity(1)

OPEN MEGA MENU
    │
    ├─► Container: opacity(0), y(10) → opacity(1), y(0)
    ├─► Category 1: delay(0ms)
    ├─► Category 2: delay(100ms)
    └─► Category 3: delay(200ms)

HOVER MENU CARD
    │
    ├─► Background: Transparent → Surface Elevated
    ├─► Icon: Primary/10 → Primary
    ├─► Arrow: opacity(0), x(0) → opacity(1), x(4px)
    └─► Text: White → Primary Hover

BUTTON INTERACTION
    │
    ├─► Hover: scale(1.0) → scale(1.05)
    ├─► Click: scale(1.05) → scale(0.95)
    └─► Release: scale(0.95) → scale(1.0)

                            STATE MANAGEMENT
═══════════════════════════════════════════════════════════════════════════════

useScrollHeader()
├─► isScrolled: boolean      (> 50px from top)
└─► isVisible: boolean       (scroll direction up)

useMegaMenu()
├─► activeMenu: string | null    (which menu is open)
├─► isOpen: boolean              (is any menu open)
├─► menuRef: RefObject          (click outside detection)
├─► openMenu()                  (open handler)
├─► closeMenu()                 (close with delay)
└─► closeMenuImmediately()      (instant close)

SearchBar State
├─► isOpen: boolean             (modal visibility)
├─► searchQuery: string         (user input)
└─► inputRef: RefObject        (auto-focus)

MobileNav State
├─► isOpen: boolean             (sidebar visibility)
├─► expandedMenu: string | null (current accordion)
└─► Body scroll lock           (prevent scroll)
